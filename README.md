# Orl_face_dataset_detection
Orl face dataset is inbuilt dataset of faces. In this dataset each folders contain 10 pictures. 

In this dataset I use SVM technique to detect the face

Requirments :

	pip install cv2
	pip install numpy
	pip install pandas
	pip install joblib
